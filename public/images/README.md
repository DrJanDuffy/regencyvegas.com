# Images for RegencyVegas.com

This directory holds all image assets for the site:

- `photos/` – property photos, community lifestyle images, amenity shots
- `logos/` – site logo, brokerage logo, partner logos

Use descriptive, SEO-friendly filenames (e.g. `regency-summerlin-clubhouse-pool.jpg`), and prefer optimized WebP/AVIF where possible.
